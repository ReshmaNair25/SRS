1)	Library Management System (LMS)
The Library Management System (LMS) is designed to automate the core functions of a library, including book cataloging, member management, book lending, and book search. The system supports two primary actors, Librarian and Member, along with a general actor, User. It securely manages transactions, maintains borrowing history, and provides quick retrieval of information. The system allows librarians to add or remove books, register new members, and track all lending activities. Members can search for books, borrow them, and return them within specified time limits. The system records every transaction to ensure accountability and timely updates of book availability.
